##switch 组件

### 描述：

弹框组件，可自定义按钮样式，文本，事件。同时页面内容，标题可依据需求定制。

### 参数说明：

| 参数     | 说明                                                     | 类型   | 默认值 |
| -------- | -------------------------------------------------------- | ------ | ------ |
| contType | 开关内容类型：('number':0/1 'char':'开/关' 'sign':'√/×') | String | 2      |

### 内容属性说明：

| 属性         | 说明             | 类型     | 默认值       |
| ------------ | ---------------- | -------- | ------------ |
| changeStatus | 改变开关状态     | function | 弹框关闭状态 |
| toggle       | 开关开启关闭切换 | function | 无           |
